# ZhouXingxing==================================================
  青城之旅 · 呼和浩特旅游景点介绍页面
==================================================
一、项目简介
-----------
本项目为"呼和浩特旅游景点介绍页面"，主题为呼和浩特市（内蒙古自治区
首府）的旅游景点与文化特色推广。页面包含景点介绍、轮播图、城市宣传片、
预约表单、动态评价加载等完整功能模块，支持丰富的视频展示和交互动画效果。

二、技术栈
-----------
- HTML5 语义化标签
- CSS3（Flexbox / Grid 布局、渐变、动画、过渡、响应式、3D变换）
- JavaScript ES6+（Class / DOM API / IntersectionObserver / 模块化）
- GSAP 动画库（ScrollTrigger 滚动触发动画）
- B站嵌入式视频播放器

三、文件结构
-----------
hohhot-tourism/          ← 项目根目录
├── index.html           ← 主页面
├── css/
│   ├── reset.css        ← CSS 重置（清除浏览器默认样式）
│   └── style.css        ← 主样式表（布局/组件/动画/响应式/详情页）
├── js/
│   ├── carousel.js      ← 轮播图模块（自动播放/手动切换/指示器）
│   ├── form.js          ← 表单验证模块（非空/格式/实时校验）
│   ├── main.js          ← 主交互模块（导航/动态内容/卡片详情/视频）
│   └── gsap-animations.js ← GSAP 动画模块（滚动触发/入场动画）
├── images/              ← 图片资源目录（使用在线占位图）
└── README.txt           ← 本说明文件

四、功能模块清单
-----------
1. 头部区域（Header）
   - Logo + 网站名称（流光扫过效果）
   - 导航菜单（6个锚点链接，悬停动画效果）
   - 移动端自适应汉堡菜单

2. 轮播图（Carousel / Banner）
   - 4 张呼和浩特景点幻灯片
   - 上一张 / 下一张按钮手动切换
   - 底部指示器点击跳转
   - 自动轮播（鼠标悬停暂停）
   - 键盘左右箭头支持
   - 浮动光点装饰效果

3. 城市宣传片区域（新增）
   - 《中国·呼和浩特》城市宣传片
   - B站嵌入式视频播放器
   - 响应式视频容器

4. 热门景点卡片区
   - 6 个景点卡片（大召寺、希拉穆仁草原、内蒙古博物院、
     昭君博物院、五塔寺、伊斯兰风情街）
   - 卡片悬浮上浮 + 图片放大 + 3D景深效果
   - 卡片点击弹出详情页面（含视频播放）
   - 详情页面包含：景点图片、视频播放器、基本信息、特色介绍、预约按钮

5. 蒙古文化特色区
   - 4 个文化专题（马头琴 & 长调、那达慕大会、呼和浩特美食、蒙古包文化）
   - 每个专题内嵌 B站 视频播放器
   - 深色渐变背景 + 金色装饰 + 浮动光点

6. 预约表单
   - 7 个字段（姓名/手机/邮箱/日期/人数/景点/留言）
   - 前端验证：非空校验、手机号格式、邮箱格式、
     日期合法性、人数范围
   - 实时错误提示（失焦校验 + 输入清除错误）
   - 提交成功反馈消息

7. 动态游客评价
   - 由 JavaScript 动态渲染评价卡片
   - "加载更多"按钮分批加载（每次4条）
   - 数据池包含8条真实风格的评价

8. 其他功能
   - 回到顶部按钮（滚动出现 / 悬停变色 / GSAP动画）
   - 滚动导航高亮（IntersectionObserver）
   - 平滑滚动到锚点
   - 按钮波纹点击效果
   - 响应式布局（适配手机/平板/桌面）
