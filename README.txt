一、项目简介
-----------
本项目为"呼和浩特旅游景点介绍页面"，主题为呼和浩特市（内蒙古自治区
首府）的旅游景点与文化特色推广。页面包含景点介绍、轮播图、城市宣传片、
预约表单、动态评价加载等完整功能模块，支持丰富的视频展示和交互动画效果。
同时包含4个文化专题子页面（美食、马头琴、那达慕、蒙古包），提供更深入的
文化体验内容。

二、技术栈
-----------
- HTML5 语义化标签
- CSS3（Flexbox / Grid 布局、渐变、动画、过渡、响应式、3D变换）
- JavaScript ES6+（Class / DOM API / IntersectionObserver / 模块化）
- GSAP 动画库（ScrollTrigger 滚动触发动画）
- B站嵌入式视频播放器

四、功能模块清单
-----------
1. 进入界面（Entrance Overlay）
   - 全屏开场动画，粒子装饰效果
   - 蒙古风格装饰图案
   - "点击进入"按钮，平滑过渡到主页面
   - 登录/注册按钮，点击弹出登录弹窗

2. 登录系统
   - 登录/注册弹窗（模态框）
   - 登录表单（用户名、密码）
   - 注册表单（用户名、邮箱、密码）
   - 默认账号：admin / 123456
   - 表单验证与错误提示

3. 头部区域（Header）
   - Logo + 网站名称（流光扫过效果）
   - 导航菜单（6个锚点链接，悬停动画效果）
   - 移动端自适应汉堡菜单

4. 轮播图（Carousel / Banner）
   - 4 张呼和浩特景点幻灯片（大召寺、草原、博物院、昭君墓）
   - 上一张 / 下一张按钮手动切换
   - 底部指示器点击跳转
   - 自动轮播（鼠标悬停暂停）
   - 键盘左右箭头支持
   - 浮动光点装饰效果

5. 城市宣传片区域
   - 《中国·呼和浩特》城市宣传片
   - B站嵌入式视频播放器
   - 响应式视频容器（深色背景 + 渐变光效）

6. 热门景点卡片区
   - 6 个景点卡片（大召寺、希拉穆仁草原、内蒙古博物院、
     昭君博物院、五塔寺、伊斯兰风情街）
   - 卡片悬浮上浮 + 图片放大 + 3D景深效果
   - 卡片点击弹出详情页面（含视频播放）
   - 详情页面包含：景点图片、视频播放器、基本信息、特色介绍、预约按钮、分享功能

7. 蒙古文化特色区
   - 4 个文化专题入口（马头琴 & 长调、那达慕大会、呼和浩特美食、蒙古包文化）
   - 每个专题内嵌 B站 视频播放器
   - 点击跳转对应专题子页面
   - 深色渐变背景 + 金色装饰 + 浮动光点

8. 预约表单
   - 7 个字段（姓名/手机/邮箱/日期/人数/景点/留言）
   - 前端验证：非空校验、手机号格式、邮箱格式、
     日期合法性、人数范围
   - 实时错误提示（失焦校验 + 输入清除错误）
   - 提交成功反馈消息

9. 动态游客评价
   - 由 JavaScript 动态渲染评价卡片
   - "加载更多"按钮分批加载（每次4条）
   - 数据池包含8条真实风格的评价

10. 其他功能
    - 回到顶部按钮（滚动出现 / 悬停变色 / GSAP动画）
    - 滚动导航高亮（IntersectionObserver）
    - 平滑滚动到锚点
    - 按钮波纹点击效果
    - 响应式布局（适配手机/平板/桌面）

九、运行方式
-----------
1. 使用本地服务器运行：
   - 方法1：python -m http.server 8080
   - 方法2：node http-server 或其他静态服务器
2. 打开浏览器访问：http://localhost:8080
3. 注意：视频播放器需要网络连接才能加载 B站 视频

十、设计特色
-----------
- 配色方案：翡翠绿（#0D4A3C）为主色调，香槟金（#C9A96E）为点缀色
- 视觉效果：流光扫过、浮动光点、3D景深、玻璃态背景
- 交互动画：卡片悬停、按钮波纹、平滑过渡、滚动触发
- 响应式设计：适配手机（320px+）、平板（768px+）、桌面（1200px+）
- 用户交互：登录系统、表单验证、模态框详情页、分享功能
